# Example SKY130 Project Skeleton

This folder shows a recommended project layout for the GUI:

- `schematics/`: xschem schematics
- `layout/`: magic layouts / exported GDS
- `netlists/`: generated schematic netlists
- `scripts/`: extraction/LVS helper scripts
- `results/`: simulation, LVS, antenna, extraction outputs
- `logs/`: runtime logs from tools

Use this structure as a starting point for real SKY130 projects.
